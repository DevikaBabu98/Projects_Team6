	Collections
	=============
	
	- List
		-Duplicate elements
		-in insertion Order
		-Iterator in both directions
		-ArrayList(sequencially in m/y,LinkedList(scattered in m/y)
	
	
	
	- Set
		-No duplicate elements
		-No order
		-Can be iteratored in one direction only
		-HashSet(random order), TreeSet(sorted in natural order)
		-HashSet can store null objects and TreeSet cant store null objects(nullpointerexception)
		-Comparison: HashSet uses equals(), TreeSet uses compareTo()
		-TreeSet: customize sorted order using Comparator interface
		
	-Map
		-HashMap,TreeMap,LinkedHashMap
		-key->value
		-TreeMap: sorted order
		-LinkedHashMap: iterate in the order in which entries were put into map
		
	-LinkedHashSet
		-no duplicate elements
		-sorted in insertion order of elements
